# 🧩 Includes do Sistema

A pasta include armazenará componentes reutilizáveis do sistema.

# 📌 Objetivo

Evitar repetição de código e facilitar manutenção do projeto.

# 📄 Arquivos Planejados

## header.php
Cabeçalho principal do sistema.

Poderá conter:
- logo
- menu
- navegação

## footer.php
Rodapé do sistema.

Poderá conter:
- redes sociais
- informações
- direitos autorais

## navbar.php
Menu de navegação principal.

## sidebar.php
Menu lateral dos dashboards.

## conexao.php
Responsável pela conexão com o banco de dados.
