# 🗄️ Modelo Inicial do Banco de Dados

O banco de dados será responsável pelo armazenamento das informações da plataforma NextMatch.

# 📌 Objetivo

Garantir organização, segurança e gerenciamento eficiente dos dados do sistema.

# 📋 Informações que serão armazenadas

- usuários
- atletas
- professores
- organizadores
- equipes
- campeonatos
- categorias
- inscrições
- rankings
- resultados
- notificações

# 👤 Tabela de Usuários

A tabela de usuários poderá armazenar:

- nome
- e-mail
- senha
- tipo de conta
- foto de perfil

# 🏆 Tabela de Campeonatos

Informações previstas:

- nome do campeonato
- modalidade
- data
- regulamento
- premiação
- categorias

# 🧑‍🤝‍🧑 Tabela de Equipes

Informações previstas:

- nome da equipe
- treinador
- atletas vinculados
- modalidade

# 📊 Objetivos Técnicos

O banco buscará:

- segurança dos dados
- rapidez nas consultas
- organização das informações
- facilidade de manutenção

# 🛠️ Banco de Dados Planejado

- MySQL
- PostgreSQL
