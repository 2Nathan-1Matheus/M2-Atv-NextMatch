# 👥 Divisão da Equipe

A divisão de tarefas da equipe ainda será definida conforme o avanço do projeto e das etapas de desenvolvimento.

Inicialmente, todos os integrantes participarão do:

- planejamento
- pesquisa
- organização do projeto
- documentação
- estruturação inicial

As futuras divisões incluirão áreas como:

- front-end
- back-end
- banco de dados
- design
- documentação
- testes
- responsividade
- funcionalidades do sistema

Todas, aguardam designação.
