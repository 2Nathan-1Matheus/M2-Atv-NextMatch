# 👥 Público-Alvo

A plataforma será destinada para usuários envolvidos com competições esportivas.

# 🏃 Atletas

Usuários que desejam:
- participar de campeonatos
- acompanhar resultados
- visualizar rankings

Faixa etária:
- 12 a 35 anos

# 🧑‍🏫 Professores e Treinadores

Responsáveis por:
- equipes
- gerenciamento de atletas
- inscrições

Faixa etária:
- 20 a 50 anos

# 🏆 Organizadores

Usuários responsáveis pela administração dos eventos esportivos.

Necessidades:
- organização
- controle de participantes
- gerenciamento de categorias
- controle de inscrições
