# 🎨 Assets do Projeto

A pasta assets será responsável por armazenar arquivos utilizados no desenvolvimento visual e funcional da plataforma.

# 📁 Estrutura

## CSS
Arquivos responsáveis pela estilização do sistema.

Exemplos:
- responsividade
- cores
- layouts
- animações

## JavaScript
Arquivos responsáveis pelas funcionalidades do sistema.

Exemplos:
- menus
- interações
- gráficos
- validações

## PHP
Arquivos responsáveis pela lógica do sistema.

Exemplos:
- autenticação
- conexão com banco
- gerenciamento de dados

## Fonts
Fontes utilizadas na identidade visual.

Fontes planejadas:
- Poppins
- Roboto
- Montserrat

## Icons
Ícones utilizados na interface do sistema.

Exemplos:
- menu
- usuário
- esportes
- notificações

# 🛠️ Tecnologias Planejadas

## Back-End
- PHP

- MySQL
